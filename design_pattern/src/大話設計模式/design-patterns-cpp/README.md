# design-patterns-cpp
《大話設計模式》中23種設計模式案例的C++實現版本。樣例忠於原書，某些地方根據C++特性做了修改。

## 組織結構
* src - 每個模式案例的聲明（.h）和實現（.cc）
* tests - 每個模式案例的gtest，相當於客戶端
* docs - 每個模式案例的UML（.html）

## 編譯結果
* patterns - src編譯得到的模式案例類庫
* patterns_run - main輸出設計模式字符圖
* patterns_test - tests中所有案例的單元測試
